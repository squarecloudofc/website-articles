## 🚀 Introdução

- Antes de começar, certifique-se de ter o Python instalado no seu sistema. Caso ainda não tenha, você pode baixá-lo no <a href={pages.python}>site oficial do Python</a>.
- Em seguida, você precisará criar uma conta na Square Cloud, o que pode ser feito através da <a href={pages.login}>página de login</a>. Você pode usar seu e-mail para criar sua conta.
- Por fim, é necessário ter um plano pago ativo na sua conta. Você pode visualizar nossos planos e adquirir aquele que melhor atenda às suas necessidades <a href={pages.plans}>aqui</a>.


## 💻 Preparando sua aplicação

### 📖 Dependências

Tenha certeza de listar todas as dependências necessárias de seu projeto em seu arquivo de dependências, seja ele `pyproject.toml` ou `requirements.txt`.
Deve conter no arquivo o Flask e o webserver usado que, neste artigo usaremos o `waitress`.

<Requirements/>

### 📂 Arquivos necessários

Enquanto prepara seu projeto para envio, certifique-se de incluir, além do seu código, o `squarecloud.app/config` e o seu arquivo de dependencia previamente preparado.
Você pode encontrar mais informações sobre como criar o `squarecloud.app/config` <a href={pages.docs.configFile}>aqui</a>. Caso envie através de nosso dashboard, não é necessário criar 
o arquivo de configuração previamente basta compactar o código e o arquivo de dependências.


### ⚙️ Configurando o webserver

Para configurar seu webserver waitress, é necessário que modifique seu código para que fique semelhante ao seguinte.

```python
from flask import Flask
# Import serve function from Waitress
from waitress import serve

# Instância da aplicação flask
app = Flask(__name__)

# Suas rotas...

# Serve a aplicação com host 0.0.0.0 (todas interfaces) e porta 80
serve(app, host='0.0.0.0', port=80)
```

## ☁️ Enviando o projeto

Compacte todos os arquivos de código necessários, arquivos de dependências e de configuração. Após isso, anexe o zip na <a href={pages.upload}>página de upload</a> 
ou envie utilizando de nossa API.
