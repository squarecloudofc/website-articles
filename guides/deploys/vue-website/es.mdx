## Introducción

- Antes de comenzar, asegúrate de tener Node.js y npm instalados en tu sistema. Si aún no los tienes, puedes descargarlos desde el [sitio oficial de Node.js](https://nodejs.org/).
- A continuación, necesitarás crear una cuenta en Square Cloud, lo cual puedes hacer a través de la [página de registro](https://squarecloud.app/signup). Puedes usar tu correo electrónico para crear la cuenta.
- Finalmente, necesitas tener un plan de pago activo en tu cuenta. Puedes ver nuestros planes y adquirir uno según tus necesidades [aquí](https://squarecloud.app/plans).

### Creando un Proyecto Vue

Para crear un nuevo proyecto Vue, ejecuta el siguiente comando:

\`\`\`bash
npm create vue@latest
\`\`\`

En este comando, \`npm create vue@latest\` configura un nuevo proyecto Vue con la versión más reciente. El comando \`npm create\` forma parte de \`npm\`, el gestor de paquetes de Node.js, y sirve para inicializar un paquete npm nuevo o existente.

\`vue@latest\` es un paquete en el registro npm que genera un proyecto Vue para ti, encargándose de la configuración inicial como la creación de directorios, instalación de dependencias y configuración de archivos básicos.

\`@latest\` es la etiqueta que garantiza que uses la versión más actualizada del paquete \`vue\`, brindándote acceso a las últimas funcionalidades.

## Configuración del Puerto 80

El puerto 80 es el predeterminado para tráfico HTTP. Para alojar una app Vite en Square Cloud, añade la opción \`--port 80\` en tu script de inicio para que tu app escuche en el puerto 80.

## Configuración del Campo START

En el archivo de configuración de Square Cloud, el campo \`START\` permite personalizar el comando de inicio. Para compilar e iniciar tu app Vue en Square Cloud, usa:

\`\`\`app squarecloud.app
START=npm run build && npm run serve
\`\`\`

Esto ejecuta los scripts “build” y “serve” de tu \`package.json\`:

\`\`\`json package.json
{
  "name": "vue-project",
  "version": "0.0.0",
  "private": true,
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "serve": "vite preview --port 80"
  },
  "dependencies": {
    "vue": "^3.4.21"
  },
  "devDependencies": {
    "@vitejs/plugin-vue": "^5.0.4",
    "vite": "^5.2.8"
  }
}
\`\`\`

## Adquirir un Plan Antes de Subir

Antes de subir tu proyecto, debes adquirir un plan. La elección depende de los recursos que tu app requiera. Consulta nuestros planes [aquí](https://squarecloud.app/plans).

<Note>**¿Te preguntas cuánta RAM y CPU necesitas para alojar un sitio Vue.js?** Vue.js es un framework progresivo conocido por su adaptabilidad y sencillez. Permite adopción incremental, renderizado declarativo y un enfoque basado en componentes. Nuestro plan **[Standard](https://squarecloud.app/pay/standard)** ofrece **4GB** de RAM y **4vCPU**, suficiente para la mayoría de sitios Vue.js. Para proyectos más grandes o mayor estabilidad, considera Pro. Para comprar, haz clic [aquí](https://squarecloud.app/pay/pro).</Note>
