## 介绍

* 在开始之前，请确保您的系统已安装 Node.js 和 npm。如果尚未安装，您可以从 [Node.js 官方网站](https://nodejs.org/) 下载。
* 接下来，您需要在 Square Cloud 上创建一个帐户，可通过 [注册页面](https://squarecloud.app/signup) 完成。您可以使用电子邮件创建帐户。
* 最后，您的帐户需要启用一个有效的付费计划。您可以在[此处](https://squarecloud.app/plans)查看我们的计划并根据需求购买。

### 设置 Express

要使用 Express，需要先安装它。运行以下命令：

```bash
npm install express
```

### 创建 Express 项目

要创建一个新的 Express 项目，请创建一个文件并写入以下代码：

```javascript app.js
// 引入 Express 模块
const express = require('express');

// 创建一个新的 Express 应用
const app = express();

// 将端口设置为 80
// 端口 80 是 HTTP 流量的默认端口。
const port = 80;

// 定义 '/' 路由的处理函数
// 访问时将返回 'Hello World!'。
app.get('/', (req, res) => {
  res.send('Hello World!');
});

// 让 Express 应用监听端口 80
app.listen(port, () => {
  // 一旦服务器在端口 80 上监听，此代码块将运行
  console.log(`Example app listening on port \${port}`);
});
```

## 配置 START 字段

在 Square Cloud 配置文件中，**START** 字段是可选的，仅在您使用自定义脚本启动网站或 API 时才需要。在示例中，**START** 字段并非必需。

## 上传项目前购买计划

在将项目上传到 Square Cloud 之前，您需要购买计划。所需计划类型取决于应用的资源需求。您可以在[此处](https://squarecloud.app/plans)查看我们的计划。

<Note>**想知道托管 Express 应用需要多少内存和 CPU？** 别担心，我们来帮您。Express.js 是 Node.js 的一个框架，提供构建 Web 服务器的最小功能。它易于学习、高度灵活，适用于各种应用。我们的 **[Standard](https://squarecloud.app/pay/standard)** 计划提供 **4GB** RAM 和 **4vCPU**，足以满足大多数 Express 应用。但如果您需要更高稳定性，建议选择 Pro 计划。更多资源可确保高峰期平稳运行。购买请点击 [这里](https://squarecloud.app/pay/pro)。</Note>

## 将项目上传到 Square Cloud

准备好项目文件后，即可将其上传到 Square Cloud 并托管。

<Tabs>
  <Tab title="通过控制台上传">
    访问 [Square Cloud 仪表板](https://squarecloud.app/dashboard/upload) 并上传项目文件。

    <Frame>
      <img src="/images/uploading.gif" style={{ borderRadius: "0.2rem" }} />
    </Frame>
  </Tab>

  <Tab title="通过 CLI 上传">
    <Steps>
      <Step title="第一步">
        需要先安装 CLI。如果尚未安装，请运行：

        ```
        npm install -g @squarecloud/cli
        ```

        如果已安装，建议更新：

        <Tabs>
          <Tab title="Windows">
            ```bash
            squarecloud update
            ```
          </Tab>

          <Tab title="Linux、macOS 和 WSL">
            ```bash
            curl -fsSL https://cli.squarecloud.app/install | bash
            ```
          </Tab>
        </Tabs>
      </Step>

      <Step title="第二步">
        获取授权密钥，请在 [帐户页面](https://squarecloud.app/account) 点击“Request API Key”，然后运行：

        ```bash
        squarecloud login
        ```
      </Step>

      <Step title="第三步">
        最后，通过 CLI 部署应用，传入 zip 文件路径：

        ```bash
        squarecloud upload zip
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>

## 故障排除

<AccordionGroup>
  <Accordion title="自定义域名" icon="globe" iconType="solid">
    如果想使用自定义域名，默认 URL 会是 `https://mysite.squareweb.app/`。子域名由 SUBDOMAIN 字段决定。若要使用 `mysite.com`，需购买 Standard 或以上计划。详见：[如何设置自定义域名](https://docs.squarecloud.app/tutorials/how-to-set-up-your-domain)。
  </Accordion>

  <Accordion title="最低内存要求" icon="memory" iconType="solid">
    托管网站或 API 的最低内存为 512MB。但根据 Express 应用的规模，可考虑更多内存以避免 [LACK\_OF\_RAM](https://docs.squarecloud.app/faqs/troubleshooting/LACK_OF_RAM) 错误。
  </Accordion>

  <Accordion title="临时访问被拒" icon="ban" iconType="solid">
