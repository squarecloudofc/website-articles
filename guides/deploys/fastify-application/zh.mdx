## 介绍

- 在开始之前，请确保系统已安装 Node.js 和 npm。如未安装，可在 [Node.js 官方网站](https://nodejs.org/) 下载。
- 接着，请在 Square Cloud 的 [注册页面](https://squarecloud.app/signup) 创建账号，可使用电子邮件注册。
- 最后，确保账户已开通付费套餐。可在 [此处](https://squarecloud.app/plans) 查看并购买符合需求的套餐。

### 配置 Fastify

安装 Fastify：

\`\`\`bash
npm install fastify
\`\`\`

### 创建 Fastify 项目

创建文件并添加以下内容：

\`\`\`javascript app.js
// 导入 Fastify 模块
const fastify = require('fastify')({ logger: true });

// 定义 '/' 路由
fastify.get('/', async (request, reply) => {
  return 'Hello World!';
});

// 监听 80 端口
fastify.listen({ port: 80 })
  .then((address) => {
    fastify.log.info(\`Example app listening at \${address}\`);
  })
  .catch((err) => {
    fastify.log.error(err);
    process.exit(1);
  });
\`\`\`

## 配置 START 字段

**START** 字段可选，仅在使用自定义启动脚本时需要。

## 上传前购买套餐

需根据应用所需资源购买套餐。套餐信息见 [此处](https://squarecloud.app/plans)。

<Note>**不确定需要多少 RAM 和 CPU？** **[Standard](https://squarecloud.app/pay/standard)** 套餐提供 **4 GB RAM** 和 **4 vCPU**，足够多数 Fastify 应用。大型项目可考虑 Pro 套餐，点击 [此处](https://squarecloud.app/pay/pro) 了解。</Note>

## 故障排除

  <Accordion title="自定义域名" icon="globe" iconType="solid">
