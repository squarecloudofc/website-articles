## Introducción

- Antes de empezar, asegúrate de tener Node.js y npm instalados. Si no, descárgalos desde la [página oficial de Node.js](https://nodejs.org/).
- Luego, crea una cuenta en Square Cloud desde la [página de registro](https://squarecloud.app/signup) usando tu correo electrónico.
- Finalmente, necesitas un plan de pago activo. Consulta y adquiere un plan según tus necesidades [aquí](https://squarecloud.app/plans).

### Configuración de Fastify

Instala Fastify:

```bash
npm install fastify
```

### Creación de un proyecto Fastify

Crea un archivo y agrega:

```javascript app.js
// Importando el módulo Fastify
const fastify = require('fastify')({ logger: true });

// Manejador para la ruta '/'
fastify.get('/', async (request, reply) => {
  return 'Hello World!';
});

// Iniciando Fastify en el puerto 80
fastify.listen({ port: 80 })
  .then((address) => {
    fastify.log.info(`Example app listening at \${address}`);
  })
  .catch((err) => {
    fastify.log.error(err);
    process.exit(1);
  });
```

## Configuración del campo START

El campo **START** es opcional y solo necesario si utilizas un script personalizado de inicio.

## Compra de un plan antes de subir

Debes comprar un plan acorde a los recursos de tu aplicación. Revisa los planes [aquí](https://squarecloud.app/plans).

<Note>**¿Cuánta RAM y CPU necesitas?** El plan **[Standard](https://squarecloud.app/pay/standard)** ofrece **4 GB de RAM** y **4 vCPU**, suficiente para la mayoría de los proyectos. Para aplicaciones grandes, considera el plan Pro [aquí](https://squarecloud.app/pay/pro).</Note>

## Solución de problemas
