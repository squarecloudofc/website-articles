## 介绍

- 在 Square Cloud 上创建一个帐户，可通过 [注册页面](https://squarecloud.app/signup) 完成。您可以使用电子邮件创建帐户。  
- 确保您的帐户有一个有效的付费计划。您可以在此查看并购买计划 [这里](https://squarecloud.app/plans)。

### 重要的网站参数
**网站端口** 必须是 **80**；如果您使用静态网站，则无需设置 `START` 参数。

在 squarecloud.app 配置文件中：
- `SUBDOMAIN`：如果您正在托管网站，请将此参数设置为所需的子域名。  
- `START`：自定义启动命令（如有需要）。

<Note>网站 URL 将为：`my-subdomain-parameter`.squareweb.app</Note>
\`\`\`systemd This example host a website without custom startup command
MAIN=index.html
MEMORY=512
VERSION=recommended
DISPLAY_NAME=我的简单静态网站！
SUBDOMAIN=my-simple-static-website
\`\`\`

\`\`\`systemd This example use all parameters and custom startup command in a website
MAIN=index.js
MEMORY=512
VERSION=recommended
DISPLAY_NAME=我的简单 Next.js 网站！
SUBDOMAIN=my-nextjs-website
START=npm run build && npm run start
\`\`\`
