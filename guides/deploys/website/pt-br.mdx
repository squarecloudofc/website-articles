### Parâmetros importantes do site
A **porta do site** precisa ser **80**; se você estiver usando um site estático, não precisa definir o parâmetro `START`.

No arquivo de configuração squarecloud.app:
- `SUBDOMAIN`: se estiver hospedando um site, defina este parâmetro para o subdomínio desejado.  
- `START`: um comando de inicialização customizado (se necessário).

<Note>A URL do site será: `meu-parametro-subdominio`.squareweb.app</Note>
\`\`\`systemd This example host a website without custom startup command
MAIN=index.html
MEMORY=512
VERSION=recommended
DISPLAY_NAME=Meu site estático simples!
SUBDOMAIN=meu-site-estatico-simples
\`\`\`

\`\`\`systemd This example use all parameters and custom startup command in a website
MAIN=index.js
MEMORY=512
VERSION=recommended
DISPLAY_NAME=Meu site Next.js simples!
SUBDOMAIN=meu-site-nextjs-simples
START=npm run build && npm run start
\`\`\`
